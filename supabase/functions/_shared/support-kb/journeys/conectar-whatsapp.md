# Conectar uma conta de WhatsApp

## Quando usar
Para enviar e receber mensagens pelo CRM e permitir que a IA responda.

## Pré-requisitos
- Celular com WhatsApp instalado e câmera funcionando.
- Acesso de admin/owner da clínica.

## Passo a passo
1. Vá em **Configurações → WhatsApp** (`/settings`).
2. Clique em **Conectar nova instância**.
3. Dê um nome interno (ex.: "Recepção", "Vendas").
4. Aguarde o QR Code aparecer (até 15s).
5. No celular, abra **WhatsApp → Aparelhos conectados → Conectar aparelho**.
6. Aponte a câmera para o QR Code do CRM.
7. Aguarde o status mudar para **Conectado** (verde).

## Como saber que deu certo
- Badge **Conectado** ao lado do nome da instância.
- Mensagens recebidas começam a aparecer no **Inbox**.

## Se algo der errado
- QR Code não aparece ou expira → `troubleshooting/whatsapp.md` (seção "QR Code").
- Conecta mas não recebe mensagens → `troubleshooting/whatsapp.md` (seção "Sem inbound").
- Erro "Sessão travada" → reconectar pela mesma tela.
